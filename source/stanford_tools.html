stanford_tools.constituency_parse(text)[source]¶
Parameters:text (String) – The text you want to parse
Returns:A list of parse trees where each tree is represented by nltk.tree.Tree. Each parse tree is associated to a sentence in the text. If one sentence in the text then the list will be of length 1.
Return type:list
stanford_tools.dependency_parse(text, dep_type='basicDependencies')[source]¶
Parameters:
  • text (String) – The text you want to parse
  • dep_type (String. Default basicDependencies) – The dependency type to use either: ‘basicDependencies’, ‘enhancedDependencies’, or ‘enhancedPlusPlusDependencies’. See for more details https://nlp.stanford.edu/~sebschu/pubs/schuster-manning-lrec2016.pdf
